## 3. Data Storage and Security

### 3.1 Data Storage
- Your data is securely stored on our backend servers with enterprise-grade hosting
- Local data is stored securely on your device using encrypted storage
- JWT tokens are stored securely using secure storage mechanisms

### 3.2 Security Measures
- **Encryption**: All data transmission uses HTTPS/TLS encryption
- **Authentication**: JWT-based token authentication for secure access
- **Access Control**: Strict access controls to protect your data
- **Regular Security Audits**: We conduct regular security assessments
- **Secure Storage**: Sensitive data is encrypted at rest

### 3.3 Data Retention
- We retain your personal information as long as your account is active
- Order history is maintained for legal and tax compliance purposes
- You can request deletion of your data at any time (see Section 7)
