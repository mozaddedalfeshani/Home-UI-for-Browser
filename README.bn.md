# আপনার ব্যক্তিগত ড্যাশবোর্ড

ভাষা: বাংলা | [Read in English](README.md)

MCLX Home UI for Browser হলো browser start page-এর জন্য একটি personal dashboard। এই ডকুমেন্টে শুধুমাত্র feature এবং বাস্তব ব্যবহারিক value নিয়ে কথা বলা হয়েছে।

## এই অ্যাপ কী করে

- আপনার জন্য clean এবং customizable home screen দেয়
- দরকারি website shortcut ও notes এক জায়গায় রাখে
- বাম পাশে AI sidebar দেয়, যেখানে task অনুযায়ী profile বদলানো যায়
- সব settings local এ save থাকে, তাই workflow বারবার সেট করতে হয় না

## সব ফিচার

### 1) Home এবং Shortcut Management

- Shortcut card add, edit, delete
- Drag and drop দিয়ে order change
- প্রতিটি shortcut-এর visit count track
- বেশি ব্যবহার করা site সামনে রাখার auto-order mode
- নতুন window-তে open করার option
- Shortcut অনুযায়ী keyboard mapping

### 2) Visual এবং Layout Customization

- Theme control (light, dark, system)
- Clock show/hide, style, position, format
- Clock color এবং glow customize
- Shortcut card size ও radius adjust
- Background image/video support
- Dynamic wallpaper mode
- Layout preset: default, compact, focus

### 3) Search Experience

- Quick search modal
- একাধিক search engine option
- Search history support

### 4) Notepad System

- Persistent note storage
- নোটের ভিতরে quick search
- এক ক্লিকে note copy
- Sticky alarm/reminder utilities

### 5) Left Sidebar AI Assistant

- Homepage-এর ভিতর integrated AI side panel chat
- Provider setup এবং AI profile controls
- Streamed AI response
- Markdown-friendly AI reply rendering
- AI reply copy করার one-click option
- AI reply সরাসরি note হিসেবে save

### 6) Left Sidebar Agents (নতুন)

- Sidebar থেকেই saved Agent profile create
- প্রতিটি Agent-এর আলাদা provider selection
- প্রতিটি Agent-এর আলাদা API key
- প্রতিটি Agent-এর আলাদা model
- প্রতিটি Agent-এর আলাদা rules
- প্রতিটি Agent-এর আলাদা language behavior
- প্রতিটি Agent-এর আলাদা tone/behavior preset
- Agent menu থেকে instant switch
- Header-এ active agent indicator
- Agent mode থেকে বের হয়ে general setup-এ ফেরা
- Refresh-এর পরও agent profile থেকে যায়

## Life-Saving Features

এগুলোই daily usage-এ সবচেয়ে বেশি কাজে লাগে।

### Fast Recovery এবং Control

- ভুল configuration হলে reset options দিয়ে দ্রুত recover
- Context switch করার সময় chat দ্রুত clear
- এক ক্লিকে Agent mode leave করে baseline setup-এ ফেরা

### Time-Saving Workflow Boosters

- দ্রুত কাজের জন্য keyboard shortcuts
- বেশি ব্যবহার করা shortcut সবসময় উপরে রাখার auto-order
- AI reply সরাসরি note-এ save (copy-paste ঝামেলা কম)
- বারবার একই rules না লিখে saved Agent reuse

### Focus এবং Context Management

- কম distraction-এর জন্য layout preset
- message-history toggle দিয়ে clean বা rich AI context control
- rules/language/behavior দিয়ে consistent output রাখা
